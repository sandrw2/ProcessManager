1. Running Instructions: Should not have to do anything besides run main, but do make sure
you have the input.txt file in the folder so it can be read.
2. Description of files:
1. Zip Folder Contents
    - "types" folder contains:
        AcquiredResource.java --> object used to represent an acquiredResource
        Request.java --> object used to represent a request for a resource
        Resource.java --> object used to represent a resource
        Process.java --> object used to represent a process
    - Simulation.java --> has all the functions used for process manager
    - Main.java --> Main class used for executing program
    - output.txt --> output file
